## Опис

Реалізація методу Ньютона для чисельного розв'язання нелінійного рівняння:

```
f(x) = x³ - 2x - 5 = 0
```

Метод Ньютона (метод дотичних) — ітераційний чисельний метод знаходження кореня функції. На кожному кроці будується дотична до графіка функції, і наступне наближення визначається як точка перетину дотичної з віссю OX за формулою:

```
x_(n+1) = x_n - f(x_n) / f'(x_n)
```
